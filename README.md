# 防晕车助手 - Android 应用

## 功能特点

### 基础功能
1. **视觉稳定器**：屏幕中心固定高亮光点，可调节大小和亮度
2. **呼吸引导**：4秒循环的呼吸节奏提示（吸气→屏住→呼气→放松）
3. **屏幕常亮**：自动保持屏幕常亮，无需频繁点亮
4. **悬浮窗模式**：支持在其他应用上方显示防晕界面

### 亮点功能
1. **陀螺仪感应**：根据手机晃动自动调整光点稳定性
2. **深色主题**：采用深蓝黑背景配高亮青色光点，减少视觉疲劳
3. **简洁界面**：极简设计，专注防晕功能

## 技术实现

- **开发语言**：Java
- **开发框架**：Android SDK
- **核心技术**：
  - WindowManager 实现悬浮窗
  - SensorManager 实现陀螺仪感应
  - PowerManager 实现屏幕常亮
  - Animation 实现呼吸动画效果

## 构建说明

### 环境要求
- Android Studio
- JDK 8 或更高版本
- Android SDK API 21+

### 构建步骤
1. 克隆项目到本地
2. 用 Android Studio 打开项目
3. 同步 Gradle 依赖
4. 构建并运行项目

```bash
# 命令行构建
./gradlew assembleDebug
```

### 权限说明
- `SYSTEM_ALERT_WINDOW`：显示悬浮窗
- `WAKE_LOCK`：保持屏幕常亮
- `VIBRATE`：触觉反馈（预留）
- `REQUEST_IGNORE_BATTERY_OPTIMIZATIONS`：忽略电池优化

## 使用方法

1. 首次使用需要授予悬浮窗权限
2. 点击"开始悬浮模式"进入防晕界面
3. 盯着中心光点，跟随呼吸引导
4. 可以拖动光点位置调整
5. 点击右上角关闭按钮退出

## 原理说明

防晕车的核心原理是通过固定的视觉参考点，减少视觉系统与前庭系统的冲突。当人在移动的交通工具中，眼睛看到的是固定的光点，而不是窗外快速移动的景物，从而减轻晕车症状。
